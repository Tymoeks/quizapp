Tworzenie nowego quizu

Prompt AI do generowania:

Wygeneruj quiz w formacie JSON na temat [TWÓJ TEMAT]. 

Parametry opcjonalne:
- Liczba pytań: [np. 10]
- Liczba pytań prawda/fałsz: [np. 3]

Quiz powinien zawierać:
- title: tytuł quizu
- Tablicę questions z pytaniami, gdzie każde pytanie ma:
  - question: treść pytania
  - correct: poprawna odpowiedź
  - wrong: tablica 3-4 błędnych odpowiedzi
  - group: kategoria pytania (np. "osoba", "data", "miejsce")
  - type: "tf" dla pytań prawda/fałsz (opcjonalne)

Przykład:
{
  "title": "Quiz o Polsce",
  "questions": [
    {
      "question": "Kto był pierwszym królem Polski?",
      "correct": "Bolesław Chrobry",
      "wrong": ["Mieszko I", "Kazimierz Wielki", "Władysław Jagiełło"],
      "group": "osoba"
    },
    {
      "question": "Warszawa jest stolicą Polski",
      "correct": "Prawda",
      "wrong": ["Fałsz"],
      "type": "tf",
      "group": "geografia"
    }
  ]
}
